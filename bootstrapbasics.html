Re-usable UI elements for Moodle (& elsewhere)

The following UI elements can be used to enhance your Moodle learning materials and create an aesthetically pleasing and consistent environment, all within the Moodle editor. They use a combination of Bootstrap (Layout, Content and Components) and FontAwesome v4.7. These are both 'baked in to Moodle' so it makes editing a breeze. Created by Greg Bird @BirdyOz

How To Use The Materials On This Page

To get the most out of the materials presented here please view the video and follow the instructions below

  • Find the element you want to use.
  • Click the 'Copy code' button
    • underneath the element.
  • Go to the area in Moodle you'd like to insert the element. This is anywhere you can access the HTML Editor (look for < > icon) via the ATTO or TinyMCE editors.
  • Paste in the code from this site that you copied.
  • Edit the template text with your directions/content.
